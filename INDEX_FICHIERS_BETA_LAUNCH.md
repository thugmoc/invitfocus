# INDEX_FICHIERS_BETA_LAUNCH

Ce fichier liste et décrit les documents opérationnels à utiliser pour le lancement BETA d’INVITEFOCUS.

### 1. `landing_page_beta_responsive.html`
Page de présentation BETA, mobile-first, prête à héberger.
- Sections: hero, features, team, CTA
- Design responsive 320-1920px
- Lien direct vers le dossier Drive client existant

### 2. `01_LANCER_BETA_START.md`
Guide de lancement BETA en 6 étapes.

- Intégration des données existantes

### 4. `03_RAPPORT_GENERATION.md`
Pipeline de génération de rapport.
- Python + Claude API + PDF

- Commandes et prompt type

### 5. `04_LAUNCH_CHECKLIST.md`
Checklist de lancement pré/post.
- 50+ items de validation
- Matrice GO/NO-GO
- Gestion des risques
- Contrôle qualité final

### 6. `05_DONNEES_CLIENT_REQUIREMENTS.md`
Requis de données client.
- Scoring qualité 0-100
- Modèles d’e-mails

- Intégration des données existantes

### 7. `06_UTILISER_VOS_DONNEES_EXISTANTES.md`
Guide d’import des clients et des documents existants.
- Processus 1-2 jours au lieu de 14+
- Priorisation des dossiers complets
- Comparaison de timing
- Conseils d’intégration

### 8. `INDEX_FICHIERS_BETA_LAUNCH.md`
Navigation de l’ensemble des fichiers BETA.
- Description de chaque ressource
- Permet une vue synthétique rapide
