- 3-5 clients onboardés
- 1ère génération de rapports
- Process validé

**Livrables**

- Airtable + Zapier configurés

## Risques principaux

- Acquisition client plus lente
- Données clients incomplètes
- Coûts API trop élevés
- Problèmes de scalabilité technique

## Actions d’atténuation

- Augmenter le marketing direct
- Process de collecte de données plus strict
- Optimiser l’usage de Claude
- Automatiser le pipeline progressivement
